# как пользоваца
есть файлек config.json, там:

HOST: айпи сервера

PORT: порт сервера

DIRECTORY: директория с файлами сервера

INDEX_FILE: главный файл (index.html, index.php и тд)

ERROR_404: путь до страницы с ашепкай 404
